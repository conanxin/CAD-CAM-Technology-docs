======================
现代 CAD/CAM 发展背景
======================

工业 4.0 中的 CAD/CAM
======================

在工业 4.0 和智能制造的背景下，CAD/CAM 技术已经从单一的设计或制造工具，演变为贯穿产品全生命周期的核心数字化技术。

关键发展趋势
============

**云化 CAD/CAM**
  Onshape、Fusion 360 等云端 CAD 平台打破了传统桌面软件的边界，支持多人实时协作和跨平台访问。

**AI 辅助设计**
  生成式设计（Generative Design）利用拓扑优化和进化算法，自动探索大量设计可能性，找到最优解。

**数字孪生 (Digital Twin)**
  将物理实体与虚拟模型实时映射，实现产品设计、制造、运维的全生命周期管理。

**增材制造集成**
  CAD 模型直接驱动 3D 打印，无需传统模具，极大缩短了从设计到实物的时间。

**MBD (Model-Based Definition)**
  以三维模型作为产品定义的唯一来源，替代传统的二维图纸，减少信息转换错误。

CAD/CAM 全流程
===============

.. image:: _static/diagrams/cad-cam-flow.svg
   :width: 100%
   :align: center
   :alt: CAD/CAM 全流程图

如上图所示，现代 CAD/CAM 系统覆盖了从概念设计到成品制造的完整流程：

1. **概念设计** → 草图、拓扑优化
2. **详细设计** → 三维建模、装配设计
3. **工程分析** → 有限元分析、运动仿真
4. **工艺规划** → CAPP、工艺路线设计
5. **数控编程** → 刀具路径生成、G 代码输出
6. **加工制造** → CNC 加工、3D 打印
7. **质量检测** → 三坐标测量、在线检测

学习建议
========

对于初学者，建议从理解 CAD/CAM 的基本概念和流程入手，逐步深入到具体建模技术和编程方法。同时，关注云化、AI 和数字孪生等前沿方向，有助于把握行业未来趋势。
