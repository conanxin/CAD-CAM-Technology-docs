==============================

学习目标
--------

2. 掌握 PDM 技术在 CAD/CAM 集成中的角色与体系结构
3. 了解柔性制造系统（FMS）、CIMS、并行工程、虚拟制造、敏捷制造、绿色制造等先进制造模式

核心概念
--------

- **CAD/CAM 集成**：在不同模块间实现数据共享与协同，消除"信息孤岛"
- **STEP 标准**：ISO 10303 产品模型数据交换标准，定义 EXPRESS 描述的产品数据模型
- **IGES（初始图形交换规范）**：早期二维/三维几何数据交换标准
- **PDM（产品数据管理）**：在产品生命周期内管理所有数据、过程和资源
- **FMS（柔性制造系统）**：由加工站、物流系统、控制系统组成的可加工多种零件的集成制造系统
- **CIMS（计算机集成制造系统）**：将设计、制造、管理等环节用信息技术集成的现代制造范式

本章导读
--------

本章作为全书的收尾，介绍 CAD/CAM 集成的核心技术与典型应用：从数据交换标准（IGES→STEP）到 PDM 系统集成方案，从柔性制造系统（FMS）到 CIMS、并行工程（CE）、虚拟制造（VM）、敏捷制造（AM）和绿色制造（GM）——勾勒出未来制造业数字化、网络化、智能化的整体图景。

8.4PDM技术集成方案
--------------------
产品数据管理（Product Data Management, PDM）技术是指以产品数据的管理为核心，通过计算机网络和数据库技术把企业
生产过程中所有与产品相关的信息和过程集成管理的一种技术。

1. 基于PDM的CAD/CAM集成系统体系结构

 基于PAM系统的CAD/CAM集成可分为三个层次，即封装、接口和集成。

2. 基于PDM的CAD/CAM集成系统的主要功能
 
 PDM系统的主要功能是用于管理在整个产品生命周期内所有与产品有关的信息。

 基于PDM的CAD/CAM集成系统所包含的主要功能有：

 （1）产品结构和配置管理。
 
 （2）图文档管理。
 
 （3）工作流程管理。
 
 （4）动态权限设置。
 
 （5）工程变更管理。
 
 （6）项目管理。
 
 （7）外部集成工具。

工程应用场景
------------

**航空发动机整机制造**
  以 STEP 标准作为统一数据模型，PDM 系统管理百万级零部件；CIMS 把产品设计、工艺规划、生产调度、质量检测集成在同一平台。

**多品种小批量柔性生产**
  汽车零部件厂用 FMS 灵活切换毛坯件品种，在同一加工单元内完成箱体类、轴类零件的自动加工与检测。

**分布式协同设计**
  采用并行工程模式，总部研发与异地设计分部通过 PDM/CSCW 平台并行开展结构、热仿真与工艺设计，显著缩短开发周期。
