# Bracket Capstone 资源包

## 用途

本资源包配合教程页面 `examples/bracket-capstone-project` 和 `examples/bracket-project-portfolio` 使用，帮助读者完成 L 型支架 Capstone 项目后系统整理项目档案和进行自我评估。

## 对应教程

- **主项目页面**: [L 型支架 Capstone 项目](https://conanxin.github.io/CAD-CAM-Technology-docs/examples/bracket-capstone-project.html)
- **项目档案**: [Bracker Capstone Project Portfolio](https://conanxin.github.io/CAD-CAM-Technology-docs/examples/bracket-project-portfolio.html)
- **评分表**: [Bracker Capstone Assessment Rubric](https://conanxin.github.io/CAD-CAM-Technology-docs/examples/bracket-assessment-rubric.html)

## 文件清单

| 文件 | 说明 |
|------|------|
| `README.md` | 资源包说明 |
| `portfolio-template.md` | 项目档案模板 |
| `submission-checklist.md` | 提交清单 |
| `reflection-template.md` | 反思笔记模板 |
| `rubric-summary.md` | 评分标准摘要 |

## 使用方法

1. 完成 L 型支架 Capstone 项目（参考 V6A）
2. 使用 `portfolio-template.md` 整理项目档案
3. 使用 `submission-checklist.md` 检查提交包完整性
4. 使用 `reflection-template.md` 撰写反思笔记
5. 参考 `rubric-summary.md` 评估自己的项目
