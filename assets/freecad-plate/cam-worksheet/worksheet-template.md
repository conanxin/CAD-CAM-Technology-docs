## 加工顺序确认

- [ ] 先面后孔（平面加工在孔加工之前）
- [ ] 先粗后精（粗加工在精加工之前）
- [ ] 换刀顺序合理（尽量减少换刀次数）
- [ ] 夹具不会与刀具干涉
- [ ] 加工顺序已优化（减少空行程）

## 安全与检查

- [ ] 刀具长度补偿已设置
- [ ] 工件坐标系（G54）已设定
- [ ] 主轴转速不超过机床上限
- [ ] 进给速度适合当前刀具和材料
- [ ] 已模拟刀路（如有条件）
- [ ] 已检查 G-code 中的 M03/M05 和换刀指令
